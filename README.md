What This Repo Is
------------------------------------------------------------
This repo is super simple — it just contains the PDF of my research paper,
“Predicting Depression Using Machine Learning: A Neural Network Approach.”

I wrote this project because I wanted to explore how machine learning can help
with early mental health screening, especially in large populations where
traditional diagnosis is slow or inaccessible.

Two lines from the paper that summarize the vibe:
“Our neural network approach achieved a 70% prediction accuracy, with a perfect recall rate (100%).”
“Income (43.17% importance) and age (25.16% importance) emerged as the strongest predictors…”

------------------------------------------------------------
Why I Did This Project
------------------------------------------------------------
As a first-year student, I wanted to push myself beyond coursework and try
building something end-to-end: data cleaning, modeling, analysis, and writing.
Mental health is also something that affects a lot of people my age, so it felt
meaningful to work on.

This project was a chance to:
- learn how to handle a large dataset (413k+ rows)
- build and tune a neural network from scratch
- practice statistical testing and interpretation
- write a full research-style paper

------------------------------------------------------------
What’s Inside the Paper
------------------------------------------------------------
The PDF covers:
- dataset description (demographic, socioeconomic, lifestyle features)
- preprocessing steps (encoding, scaling, splitting)
- neural network architecture (256 → 128 → 64, dropout, ReLU)
- model performance (70% accuracy, 100% recall)
- statistical tests (chi-square, t-tests, ANOVA)
- feature importance (income and age dominate)
- odds ratios for lifestyle factors
- discussion of limitations and ethics

Even though I’m early in my degree, I tried to make the analysis clear,
transparent, and reproducible.
